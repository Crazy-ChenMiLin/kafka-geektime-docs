## 爬虫经验总结

### 遇到的问题

1. **URL 编码问题**
   - 手动构造的 URL 很多返回 404
   - 中文字符需要正确的 URL 编码

2. **章节编号不连续**
   - 专栏章节不是连续的 1-42
   - 实际只有部分章节：01-06, 18, 23, 26, 29, 32, 33, 36-42

3. **链接提取困难**
   - 需要从首页 HTML 中提取真实链接
   - 使用正则表达式解析导航菜单

### 解决方案

1. ✅ 先抓取首页获取真实的导航链接
2. ✅ 从 HTML 中提取所有有效的章节 URL
3. ✅ 批量下载这些真实存在的页面
4. ✅ 清理重复文件

### 技术栈

- PowerShell 脚本
- WebFetch API
- GitHub MCP
